# Dessert Release

The Dessert Release app is a simple app which displays a collection of Android dessert releases. 
This app is used to teach the concept of Preference DataStore.

tk

Environment

- Kotlin 1.8.20
- Android Studio Flamingo 2022.2.1
- Gradle Plugin 8.0.0
